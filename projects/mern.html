SecretKeeper App

SecretKeeper is a project I built using MERN stack. Users can post anonymous secrets, like others' posts, and manage their own content. I used GraphQL to handle all data requests through a single endpoint and JSON Web Tokens for user authentication. My backend is built with Node and Express, and everything connects to MongoDB for data storage.
